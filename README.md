# Job Salary Analytics

## Overview

This project contains code that provides a comprehensive analysis of employment and
recruitment trends using data from LinkedIn job postings and the Bureau of Labor Statistics (BLS). Over 124,000 job listings
from 2023-2024 were analyzed, focusing on job status, wages, recruitment trends,
and labor participation. The study combines multiple data sources to provide insights into current labor market dynamics, salary
transparency, and evolving workplace arrangements. The findings reveal significant patterns in employment structures, compensation
practices, and the increasing prevalence of remote work options.

## Results

* Job postings tend towards non-transparent salary disclosure and off-site applications on LinkedIn
* Retail, Wholesale, and Finance sectors work more for less pay while Education sectors work more for more pay
* Machine Learning analysis indicated that are significant discrepancies between salary ranges given by LinkedIn postings in comparison to the standard salaries for those positions as indicated by governmental data

## Data Sources

* LinkedIn Job Postings
* Current Employment (BLS)
* Occupational Employment and Wage (BLS)
* Short-Term Labour Force (ILO)

## Architecture

* Data Ingestion
* Data Processing with Apache Spark
* Query based analysis
* Machine Learning with MLLib
* Visualization with Tableau
